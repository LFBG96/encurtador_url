<h1>Encurtador de url</h1>

<p>Este projeto tem como finalidade encurtar uma url</p>
<p>caso você precise encurtar uma url muito longa</p>

<h3>foi usada a biblioteca externa Flask</h3>

<p>Esse projeto não tem conexão com banco de dados sql, mas guarda os links em um arquivo .json chamado urls.json</p>
<p>Para fazer um projeto funcionar você deve apenas apertar F5 ou clicar em run da sua IDE</p>
<p>Coloque o link http://localhost:5000 em seu navegador, e ele abrirá uma pagina com um input para por sua url e um botão, para gerar a url curta</p>
